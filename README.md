# 贵州菜价可视化系统

## 项目概述

贵州菜价可视化系统是基于 Python 爬虫技术，使用 Django 框架开发的一个数据展示平台。系统致力于通过数据爬取、处理和可视化，为用户提供直观的贵州菜价信息。

## 技术栈

- 开发语言：Python
- 数据库：MySQL
- Web框架：Django
- 开发工具：PyCharm, Navicat

## 功能模块

- 用户登录
- 管理员主界面
- 菜价列表
- 数据统计分析

## 系统角色

- 普通用户：可查看菜价信息
- 管理员：管理用户及菜价信息，进行数据统计分析

## 运行环境

- Python环境
- MySQL数据库环境

## 部署步骤

1. 安装Python及相关依赖
2. 安装MySQL数据库
3. 配置数据库连接
4. 部署Django项目
5. 运行项目
